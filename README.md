TheaterTickets
==============
Η παραπάνω εργασία με τίτλο "Κράτηση θέσεων Θεάτρου" έγινε από τους:

Παναγιώτης - Νεκτάριος Δημητρακόπουλος 39275
Γιώργος Χαλικιόπουλος 37082
Χρήστος Πετρόπουλος 40202

Το πρόγραμμα και τα αρχεία είναι γραμμένα σε γλώσσα C και χτισμένα για Windows XP+ και Windows Server.
